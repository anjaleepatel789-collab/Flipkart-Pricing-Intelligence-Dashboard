# Flipkart Pricing Intelligence Dashboard

## About This Project
While exploring e-commerce data, I wanted to understand something specific — are brands that offer higher discounts actually selling better quality products, or is it just a pricing game?

This project analyzes 20,000 real Flipkart product listings to uncover pricing patterns, discount strategies, and FK Advantage product distribution across categories.

## Dataset
- **Source:** Kaggle — Flipkart E-Commerce Dataset
- **Size:** 20,000 rows, 15 columns
- **Type:** Real product listing data

## What I Did
- Cleaned messy raw data — removed irrelevant columns, extracted main categories from nested text using Excel formulas
- Calculated discount % for each product
- Built pivot tables to analyze patterns across categories and brands
- Created an interactive dashboard with slicers for dynamic filtering

## Key Findings
- Clothing is the largest category — 6,198 out of 20,000 products
- Sunglasses offer the highest average discount — 59%
- Gaming follows at 48% average discount
- Only 785 products (4%) carry the FK Advantage label
- eBooks have the lowest average discount — just 8%

## Business Insight
High discounts don't always mean high quality. Categories like Sunglasses and Gaming show aggressive discounting — suggesting brands use discounts as a visibility strategy rather than a quality signal.

## Skills Used
- Data Cleaning — removing duplicates, handling blanks, text extraction
- Excel Formulas — IF, IFERROR, MID, FIND, SUBSTITUTE, TRIM
- Pivot Tables — multi-dimensional analysis
- Conditional Formatting — visual pattern identification
- Interactive Dashboard — slicers, KPI cards, charts
